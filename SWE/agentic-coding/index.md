# Agentic coding

Working with AI coding agents — the tools, their runtimes, and the practices they
change.

## Subdirectories

- [agentic-loop](/SWE/agentic-coding/agentic-loop/index.md) — the core execution loop of an LLM agent (reason→act→observe) and the "loop engineering" lexicon around it
- [claude-code](/SWE/agentic-coding/claude-code/index.md) — Anthropic's Claude Code agent: CLI, cloud runtime, configuration
- [code-context](/SWE/agentic-coding/code-context/index.md) — tools that build and serve structured codebase context (knowledge graphs, indexes) to coding agents
- [context-engineering](/SWE/agentic-coding/context-engineering/index.md) — structuring, curating, and managing LLM conversation/agent context
- [frameworks](/SWE/agentic-coding/frameworks/index.md) — libraries for building and orchestrating agents
