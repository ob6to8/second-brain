# analysis (type)

The controlled `type` for a point-in-time evaluation or decision-support write-up: a question investigated against evidence (often the live bundle itself), yielding findings and a recommendation. Distinct from a `plan` (intended work) and a `note` (an idea). Defined in the [controlled-type vocabulary](/meta/policy/controlled-type-vocabulary.md).
