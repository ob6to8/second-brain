# belief network

A probabilistic graphical model, also called a *Bayesian network*: a directed acyclic graph whose nodes are stochastic (random) variables and whose edges encode conditional dependence, so each variable's distribution is defined given its parents. Popularized by Judea Pearl, the "belief" is the graphical-models sense — a probabilistic dependency structure — with no connotation of confidence or opinion. A [deep belief network](/beliefs/glossary/deep-belief-network.md) is a deep, learned instance of this idea. (Contrast the epistemic-grounding graph of the brain's [epistemic overlay](/beliefs/glossary/epistemic-overlay.md), whose edges carry no probabilities.)
