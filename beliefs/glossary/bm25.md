# BM25

A lexical full-text ranking function that scores documents against a query using term frequency and inverse document frequency, normalized for document length. It ranks by keyword relevance without any notion of meaning, making it the standard middle tier between plain text matching (grep) and [semantic search](/beliefs/glossary/semantic-search.md) — cheap, deterministic, and effective until vocabulary mismatch demands [embeddings](/beliefs/glossary/embeddings.md).
