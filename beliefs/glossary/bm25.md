# BM25

It knows nothing of meaning — pure keyword overlap — which places it one tier above plain grep and below [semantic search](/beliefs/glossary/semantic-search.md). Cheap and deterministic, it holds up until vocabulary mismatch (a query and a relevant document sharing no words) forces the jump to [embeddings](/beliefs/glossary/embeddings.md).
