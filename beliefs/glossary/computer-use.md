# computer use

Universality is the strength and the weakness at once: any pixel-rendering
program can be operated, but nothing comes back except more pixels — no return
values, no structured errors, no addressable state. It sits at the opposite
pole from [remote procedure call](/beliefs/glossary/remote-procedure-call.md)-style
process control, and between the two poles lies the historically skipped middle
layer: structured control of the running app the human is watching.
