# constant tutorial mode

The reader state in which the operator of an agent-built system is permanently mid-tutorial: systems are being built through natural-language collaboration that the operator does not yet fully understand, so the operator is learning how the system works *by using it*. In this state, dense technical references easily lose their referents, and even a glossary leads into nested webs of concepts where the intuition is lost. The plain tier of [three-level documentation](/beliefs/glossary/three-level-documentation.md) is designed as this reader's fail-safe backstop — a place to recover the general thrust and meaning when the technical register fails to land.
