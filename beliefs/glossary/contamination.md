# contamination

The classic route is training data that included the benchmark; the agent-era route is plainer — a [gold set](/beliefs/glossary/gold-set.md) committed to the same repository the agent operates on is an answer key the examinee can open and read. Tolerable when trending a fixed mechanical layer against its own past; invalidating for any comparative claim. The remedy is a [held-out set](/beliefs/glossary/held-out-set.md).
