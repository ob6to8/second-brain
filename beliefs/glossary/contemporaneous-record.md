# contemporaneous record

Regulated lab practice engineers the property physically: bound notebooks, numbered pages, ink, dated entries, no blank space left to backfill — making backdating awkward by construction. The digital analogs are prompt commits to version control and write-once fields like this bundle's [attribution](/beliefs/glossary/attribution.md), which give a file the provably-existed-then quality paper gets from physics.
