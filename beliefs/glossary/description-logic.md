# description logic

A family of deliberately restricted first-order fragments engineered for
reasoning over concepts (classes), roles (relations), and individuals — chosen
so that the standard reasoning tasks (consistency, subsumption,
classification, instance retrieval) are decidable, trading expressiveness for
guaranteed termination. The formal basis of
[OWL](/beliefs/glossary/owl.md): OWL 2 DL's Direct Semantics aligns with the SROIQ
description logic, which is why decades of DL reasoning research apply to web
ontologies directly.
