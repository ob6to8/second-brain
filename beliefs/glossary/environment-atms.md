# environment (ATMS)

A set of [assumptions](/beliefs/glossary/assumption.md), read logically as their
conjunction. Environments are what the ATMS labels each belief with (the sets
of assumptions under which it holds), and an environment is *inconsistent*
when falsity is derivable from it. The closure of a consistent environment —
its assumptions plus everything derivable from them — is an ATMS *context*:
one coherent worldview, of which many coexist. This is the canonical answer to
comparing internally-coherent but mutually-contradictory belief systems
without merging them into one inconsistent store.
