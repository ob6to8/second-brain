# epistemic overlay

A structural layer added above a set of knowledge documents that classifies each by its epistemic role (e.g. attestation, aggregation, inference, prescription) and links them into a network of conceptual dependencies. It enables cross-resource explorability and machine-checkable integrity without atomizing the underlying documents — the shape Composable Beliefs imposes and a candidate overlay for this brain.
