# epistemic overlay

The epistemic roles are things like attestation, aggregation, inference, and prescription. It does this without atomizing the underlying documents — the shape Composable Beliefs imposes, and a candidate overlay for this brain. Under concurrent autonomous writers the overlay stops being an enhancement: "what grounds this / what rests on this?" becomes the query an acting agent must ask, and [staleness propagation](/beliefs/glossary/staleness-propagation.md) traverses its edges.
