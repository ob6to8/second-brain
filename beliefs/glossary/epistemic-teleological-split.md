# epistemic/teleological split

The distinction between *epistemic* statements — claims about how the world is ("I think the world works this way") — and *teleological* statements — standing directions for action ("given that, this is the direction we consistently move"). The split sharpens the boundary between a [decision prior](/beliefs/glossary/decision-prior.md) and a [doctrine](/beliefs/glossary/doctrine.md): doctrine is not belief with more confidence, since two reasoners can share a belief and hold opposite doctrines.

**In this brain:** a filing test — epistemic statements file as `claim`/`concept`, standing directions as `doctrine` — which the [belief-layer plan](/meta/plans/belief-type-and-beliefs-namespace.md) proposes extending with a `belief` type for value-laden priors between the two.
