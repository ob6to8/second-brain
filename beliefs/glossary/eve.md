# eve

Positioned as "Next.js for agents" and integrated with Vercel's deployment primitives (AI Gateway, Sandboxes, Workflows), it is the exemplar of the continuous-automated-eval model: [regression testing](/beliefs/glossary/regression-testing.md) transplanted onto agent behavior. The bundle's analysis reads it as all-harness-no-ledger — trend lines without [adjudication](/beliefs/glossary/adjudication.md) or lineage records underneath, and test suites living inside the directory the agent itself reads.
