# excerpt log (route-tagged log)

The generated `## Thread excerpts — route-tagged log` section a
[sink](/beliefs/glossary/route-tag-sink.md) carries: append-only, per-thread,
date-stamped blocks, each lifting the [route-tagged](/beliefs/glossary/route-tag.md)
regions of one thread whole (ATX headers demoted to bold). It is the
[materialized](/beliefs/glossary/materialize.md) output of the tags — written by
`mix brain.route_tags --materialize`, guarded by the `log fidelity` check that
re-derives each block and fails on divergence, and never hand-edited.
Canonically defined by the
[route-tagging policy](/meta/policy/route-tagging.md).
