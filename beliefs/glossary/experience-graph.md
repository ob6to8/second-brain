# experience graph

Named by the EXG paper, captured as
[EXG: Self-Evolving Agents with Experience Graphs](/knowledge/SWE/agentic/agent-memory/experience-graphs-exg.md).
Where a [decision graph](/beliefs/glossary/decision-graph.md) records *how a
system decided*, an experience graph organizes *what its attempts taught it*;
both belong to the structured-agent-memory family whose measured advantage over
raw-trajectory memory is distilled, linked records.
