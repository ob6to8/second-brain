# graduation

An entry has outgrown the glossary when it accumulates senses, claims, or citations that belong in the domain tree (e.g. `/knowledge/SWE/…`); the stub takes the shape of a [pointer entry](/beliefs/glossary/pointer-entry.md), so old glossary links keep resolving. The glossary is a staging layer, not a terminal home. Defined by the [`/add-to-glossary` skill](/.claude/skills/add-to-glossary/SKILL.md).
