# lede

Because the site generator places it directly above the markdown body, whatever the body's first paragraph says sits immediately under it on the rendered page — the stacking that makes restated definitions so visible, and the reason the glossary's [single-overview convention](/beliefs/glossary/single-overview-convention.md) exists.
