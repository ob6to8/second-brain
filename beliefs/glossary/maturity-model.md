# maturity model

The genre to which Boris Cherny's
[Steps of AI Adoption](/knowledge/SWE/agentic/adoption/steps-of-ai-adoption.md) belongs:
five levels from gated access to AI-native operation, each defined by the human's role
and the agent count they run. Its analytical value is showing that the constraint
*migrates* level to level — attention, then review throughput, then decision throughput
and trust, then work identification — so the tooling that unblocks one stage is obsolete
at the next. A standing caution applies when the author is also a vendor: the level
descriptions double as a product-adoption funnel, so the bottleneck-and-lever analysis
can be sound while the mapping of products to stages reads as one vendor's instantiation.
