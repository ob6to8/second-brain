# non-bundle namespace

A top-level directory excluded from the identity registry, so its docs carry no `em:` ids and are skipped by the verifier (like `meta/` and `inbox/`), as opposed to the bundle concepts that must be identified. Explained in the [bundle-scope tutorial](/meta/tutorials/bundle-scope-and-non-bundle-namespaces.md).
