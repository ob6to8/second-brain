# operating contract

The root `CLAUDE.md` that every agent auto-loads at session start, codifying the brain's operative rules. It is a [compiled contract](/beliefs/glossary/compiled-contract.md) — generated from [`meta/preamble.md`](/meta/preamble.md) and the policies under `meta/policy/` via `mix brain.contract`, never hand-edited. It binds *agents*, not the operator, who authors and ratifies its rules without being subject to them; and "contract" names this role, not the file — `CLAUDE.md` is only the harness [mounting point](/beliefs/glossary/mounting-point.md) where the role is delivered. See [`/CLAUDE.md`](/CLAUDE.md).
