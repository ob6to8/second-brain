# preregistration

Borrowed from clinical trials and the social-science replication reforms, where cherry-picking flattering outcomes is the known failure. In a git-based eval setup the mechanism is nearly free: a committed gold doc and scoring spec whose history proves what was declared before any run — the same anti-backdating property as this bundle's write-once [attribution](/beliefs/glossary/attribution.md).
