# query profile

The set of per-category interest signals derived by walking a knowledge base's taxonomy (its `index.md` tree plus concept titles and tags), used as the retrieval filter for finding candidate items — so the taxonomy itself becomes the query. In this brain it drives the `/research` candidate feed.
