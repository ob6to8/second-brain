# sink (route-tag sink)

The document that a [route tag](/beliefs/glossary/route-tag.md) routes into and that accretes the aggregated excerpt log. An aggregating reference is a concept's stable `sb:` id (it accretes a log block); a code or file path is a non-aggregating back-link that carries no log. The sink freezes acceptance of new excerpt blocks when its matter resolves.
