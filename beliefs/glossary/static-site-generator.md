# static-site generator

A tool that renders source files (typically markdown with YAML frontmatter) into a complete set of pre-built static HTML/CSS/JS pages, served without any running application backend. This brain's `mix brain.site` is a dependency-free one that publishes the OKF bundle to GitHub Pages.
