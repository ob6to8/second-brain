# Tree-sitter

The typical front-end pipeline: parse files into syntax trees, then walk those trees to extract definitions, calls, and imports — the raw structure that gets assembled into a [code knowledge graph](/beliefs/glossary/knowledge-graph.md). Both GitNexus and Codebase-Memory build on it (66 languages in the latter).
