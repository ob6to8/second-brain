# tutorial (type)

The controlled `type` for a long-form explanatory note meant to be read start to finish (the why/how behind the tooling or a topic), distinct from a terse `note` and from a `reference` capture of external material. Lives under `meta/tutorials/`. Defined in the [controlled-type vocabulary](/meta/policy/controlled-type-vocabulary.md).
