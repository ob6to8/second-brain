# Index Spec

`index.json` is a derived cache generated by `scripts/build-index.sh`. Files on disk are the source of truth.

## Structure

```json
{
  "sources": [...],
  "assertions": [...],
  "edges": [...]
}
```

## Source Entries

| Field | Type | Description |
|---|---|---|
| `id` | string | Filename without extension |
| `type` | string | Always `source` |
| `source_type` | string | `article`, `video`, `paper`, `journal`, `transcript` |
| `title` | string | Source title |
| `tags` | string[] | Tags array |
| `file` | string | Relative path to file |

## Assertion Entries

| Field | Type | Description |
|---|---|---|
| `id` | string | Filename without extension |
| `type` | string | Always `assertion` |
| `tier` | string | `primitive` or `compound` |
| `claim` | string | The assertion claim |
| `confidence` | string | `high`, `medium`, `low`, or `contested` |
| `tags` | string[] | Tags array |
| `deps` | string[] | Dependency IDs (compounds only) |
| `file` | string | Relative path to file |

## Edge Entries

Edges are derived from frontmatter fields. Each edge:

| Field | Type | Description |
|---|---|---|
| `from` | string | Source node ID |
| `to` | string | Target node ID |
| `type` | string | `evidence`, `dependency`, or `supersedes` |

### Edge Derivation

- **evidence**: from assertion `evidence[].source` wikilinks → `{ from: assertion-id, to: source-id, type: "evidence" }`
- **dependency**: from compound `deps[]` wikilinks → `{ from: compound-id, to: dep-id, type: "dependency" }`
- **supersedes**: from assertion `supersedes` wikilink → `{ from: new-id, to: old-id, type: "supersedes" }`

## Regeneration

```bash
./scripts/build-index.sh
```

Scans `sources/*.md` and `assertions/*.md`, extracts frontmatter via `yq`, builds the index. Warns on broken wikilinks (referenced ID doesn't exist as a file).
