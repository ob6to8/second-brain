# dependency-free

A property of a program that relies only on its language/runtime standard library and pulls in no external packages, so it needs no package-fetch step and runs offline in restricted or ephemeral (e.g. no-egress, snapshot-booted) environments. This brain's Elixir tooling is deliberately dependency-free for exactly this reason.
