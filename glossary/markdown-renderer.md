# markdown renderer

A component that parses markdown source and emits HTML, handling constructs such as headings, nested lists, tables, fenced code, blockquotes, inline emphasis, and links (including link/anchor rewriting). This brain's `SecondBrain.Markdown` is a dependency-free one feeding the site generator.
