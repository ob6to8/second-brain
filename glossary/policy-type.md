# policy (type)

The controlled `type` for a standing governance rule about how the brain operates — the layer that binds every agent session. Policy documents live under `meta/policy/` and are the source from which the [operating contract](/glossary/operating-contract.md) (`CLAUDE.md`) is compiled (`mix brain.contract`). Distinct from a [plan](/glossary/plan-type.md) (a one-off intended change), an [issue](/glossary/issue-type.md) (a problem to track), and a [methodology](/glossary/methodology-type.md) (a repeatable how-to): a policy is a durable *rule*, not a task or a procedure. Defined in the [controlled-type vocabulary](/meta/policy/controlled-type-vocabulary.md).
