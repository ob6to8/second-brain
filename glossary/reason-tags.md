# reason tags

The governed vocabulary — `recent` · `impactful` · `influential` · `groundbreaking` · `buzz` — marking *why* each `/research` inbox candidate was surfaced (distinct from a [route tag](/glossary/route-tag.md), which marks which concept a thread region feeds). Defined by the [`/research` skill](/.claude/skills/research/SKILL.md).
