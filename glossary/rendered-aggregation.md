# rendered aggregation

A derived document generated by a compiler task from scattered canonical sources — an aggregation with one direction of truth: the sources are edited, the view is only ever re-rendered, and a drift gate (a byte-diff `--check` or re-derivation check) fails CI when they diverge. This brain has five: the [operating contract](/glossary/operating-contract.md) (`CLAUDE.md`, a [compiled contract](/glossary/compiled-contract.md)), the id registry (`meta/registry.md`), the route-tagged excerpt logs inside concepts, the GitHub Pages site, and the ephemeral [session-init digest](/glossary/session-init-digest.md). Among them only the contract carries *authority* — it binds agent behavior; the others index, quote, present, or orient — which is why "contract" stays a deliberate singleton within the family.
