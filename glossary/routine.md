# Routine

In the Claude Code Remote scheduling system, a Routine is a saved scheduled trigger that automatically fires a preset prompt into an agent session on a cron schedule. It can resume an existing session or spawn a fresh one on each firing, letting recurring agent work (like the daily `/news` inbox) run unattended.
