# Inbox — daily candidate feed

The brain's **waiting room**: a daily, categorized feed of research, articles, papers,
and resources matched against the [taxonomy](/index.md), each with a one-line
synopsis and a reason tag (`recent` · `impactful` · `influential` · `groundbreaking`
· `buzz`). Generated by [`/research`](/.claude/skills/research/SKILL.md).

Items here are **candidates, not concepts** — this namespace sits *outside* the OKF
bundle (no `em:` ids, never verified), the same way `meta/` does. A candidate enters
the brain only when the operator says *"intake the &lt;…&gt; item"*, which runs
[`/intake`](/.claude/skills/intake/SKILL.md) to fetch, distill, and file it.
