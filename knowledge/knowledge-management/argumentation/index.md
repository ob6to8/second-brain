# argumentation

The structure of arguments — how claims, evidence, and the licenses connecting
them are anatomized, diagrammed, and audited.

## References

- [The Toulmin model of argument](/knowledge/knowledge-management/argumentation/toulmin-model-of-argument.md) — claim / grounds / warrant / backing / qualifier / rebuttal; justification over inference as argument's first-class object.
- [Assurance cases and Goal Structuring Notation (GSN)](/knowledge/knowledge-management/argumentation/assurance-cases-and-gsn.md) — arguing a system property as an explicit claim tree down to evidence leaves; the safety-critical engineering practice.
- [Dung's abstract argumentation frameworks](/knowledge/knowledge-management/argumentation/dung-abstract-argumentation-frameworks.md) — arguments + an attack relation as a directed graph; acceptability semantics (grounded/preferred/stable) compute which sets can rationally co-stand — the formal engine of consensus/conflict evaluation.
